## One-Click Build & Install

```powershell
# Open PowerShell 7 as Administrator, navigate to the project folder:
cd .
.\install.ps1
```

`install.ps1` automatically: cleans → generates strong-name key → builds → registers COM → restarts taskbar.

After installation, right-click the taskbar → **Toolbars** → check **"DeepSeek DeskBand"**, then left-click the widget → Set API Key.

---

## Full Uninstall

```powershell
# Run as Administrator:
.\uninstall.ps1
```

`uninstall.ps1` cleans **all 4 items**:

| Step | Clears |
|------|--------|
| COM unregister | `RegAsm /unregister` |
| Registry cleanup | `HKCR\CLSID\{...}` & ProgID |
| **API Key removal** | Windows Credential Manager `DeepSeekDeskBand:ApiKey` |
| Taskbar refresh | Restart Explorer |

> **!** Deleting files alone is not enough! You must run `uninstall.ps1` to remove COM registration and credentials.

## MSI Installer (Recommended)

One-click install, supports uninstall via **Settings → Apps → Apps & features**.

### Build MSI

```powershell
# WiX Toolset is required; the script will install it automatically
.\installer\build-installer.ps1
```

Output: `installer\DeepSeekDeskBand.msi`

### Install

1. Double-click `DeepSeekDeskBand.msi` → Next → Install
2. Right-click taskbar → **Toolbars** → check **"DeepSeek DeskBand"**
3. Left-click the widget → **Set API Key**

### Uninstall

**Settings → Apps → Apps & features → DeepSeek DeskBand → Uninstall**

> **!** The API Key in Credential Manager is cleared automatically on uninstall, but you can manually verify:
> ```batch
> cmdkey /list | findstr DeepSeek
> ```

### MSI Advantages

| Feature | MSI Installer | install.ps1 |

| Automatic COM cleanup on uninstall | Yes | Yes |

| Silent/bulk deployment | Yes (`msiexec /i`) | No |

## Manual Build (without install.ps1 / MSI)

```batch
# 1. Generate strong-name key (once only)
sn -k StrongName.snk

# 2. Build
dotnet build -c Release

# 3. Register COM as Administrator
C:\Windows\Microsoft.NET\Framework64\v4.0.30319\RegAsm.exe bin\Release\net481\DeepSeekDeskBand.dll /codebase

# 4. Restart Explorer
taskkill /f /im explorer.exe && start explorer.exe
```

Manual uninstall:
```batch
C:\Windows\Microsoft.NET\Framework64\v4.0.30319\RegAsm.exe bin\Release\net481\DeepSeekDeskBand.dll /unregister
cmdkey /delete:DeepSeekDeskBand:ApiKey
taskkill /f /im explorer.exe && start explorer.exe
```

## Setting Up API Key

1. Left-click the widget on the taskbar → **Set API Key...**
2. Enter your DeepSeek API Key (`sk-...`)
3. Click **Test Connection** to verify
4. Click **Save** → Key is stored encrypted in Windows Credential Manager
